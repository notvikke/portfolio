**High Frequency Crypto Trade Simulator** is a robust microservices-based trading ecosystem designed for ultra-low latency and high scalability. It leverages the performance of Rust for core execution and the resilience of Kubernetes for orchestration.

<!--more-->

## 🚀 Key Features

*   **High-Performance Engine**: Core components written in **Rust** for minimal latency and high throughput.
*   **Scalable Infrastructure**: Fully containerized with **Docker** and orchestrated using **Kubernetes** and **Helm**.
*   **Secure Architecture**: Integrated **HashiCorp Vault** for enterprise-grade secret management.
*   **Advanced Data Storage**: Utilizes **TimescaleDB** for efficient handling of massive time-series market data.
*   **Real-Time Dashboard**: A modern, responsive **React** (Vite) interface for monitoring automated trading strategies and system health.
*   **Event-Driven Design**: Powered by **Redpanda** for reliable data streaming and service decoupling.

## 🛠️ Tech Stack

*   **Languages**: Rust, TypeScript, SQL
*   **Infrastructure**: Kubernetes, Helm, Docker, HashiCorp Vault
*   **Data & Messaging**: TimescaleDB, Redpanda
*   **Frontend**: React, Vite, Tailwind CSS
