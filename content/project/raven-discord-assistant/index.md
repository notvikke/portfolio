A bridge between Discord and high-performance AI command-line interfaces (CLIs), specifically Gemini CLI and Claude Code. Raven transforms your Discord into a remote command center for autonomous coding, environment setup, and system automation.

<!--more-->

### Key Features
- **AI-Powered CLI Bridge**: Command your PC via Gemini and Claude Code directly from Discord.
- **Autonomous Coding**: Build, debug, and refactor code remotely.
- **File System Mastery**: Read, write, and reorganize project structures via AI.
- **Spotify Integration**: Control your music playback with custom Discord commands.
- **Secure & Private**: Isolated "Home" directory and owner-only enforcement.
